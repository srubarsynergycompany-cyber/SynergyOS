# Codex Task Template

Use this template for implementation tasks in SynergyOS.

## Context
Read `.codex/PROJECT_CONTEXT.md` first.
Then read the relevant files in `docs/ai` before changing code.

## Task
Describe the exact feature or fix.

## Rules
- Keep the change small.
- Do not modify unrelated files.
- Do not change production behavior unless requested.
- Prefer safe MVP implementation.
- No external API calls unless explicitly approved.
- Show changed files before commit.

## Output Required
- Summary of changes.
- Files changed.
- How to test.
- Risks or follow-up work.
